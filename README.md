# Payment Tracking Dashboard

## Problem Statement
Hotels process multiple payment transactions daily. Managing these records manually can lead to errors and inefficient tracking. This dashboard provides a centralized system for payment management and revenue monitoring.

## Features
- Add Payment
- Delete Payment
- Search Payment
- Filter Payments
- Revenue Statistics
- Local Storage Support

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Local Storage API

## Future Scope
- Database Integration
- Authentication System
- Payment Gateway Integration
- Analytics Dashboard
